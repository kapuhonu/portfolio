Our portfolio site runs on Jekyll, with development aided by Gulp.

## Prerequisites

* Ruby
* NodeJS & NPM

## Usage

`gem install jekyll`
Run `npm install` in the project folder.

To begin development, simply run `gulp` in the project folder.
