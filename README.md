Our portfolio site runs on Jekyll, with development aided by Gulp.

## Prerequisites

* Ruby
* NodeJS & NPM

## Usage

#### 1. Install Jekyll

```sh
$ gem install jekyll
```

#### 2. Install Gulp globally

```sh
$ npm install --global gulp
```

#### 3. Install project dependencies

Navigate to the project folder, and run:

```sh
$ npm install
```
#### 4. Run Gulp

```sh
$ gulp
```
